1. Install Python 3
   https://www.python.org/downloads/

2. Install PyQt5
   http://???
   
3. Install PyYaml
   http://pyyaml.org/wiki/PyYAMLDocumentation
   http://stackoverflow.com/questions/14261614/how-do-i-install-the-yaml-package-for-python

3. Execute "imperialism_remake/start.py" in this folder with Python 3.
   e.g. python3 start.py
   
On the first start a sub folder "Imperialism Remake User Data"
in the current user home folder is created. It contains
preferences and logs.
